# sycommunity

## 바이브 코딩 놀이터

초등 수업에 활용할 수 있는 교과 웹앱을 한곳에 모아두는 개인 홈페이지입니다.

## 주요 기능

- 홈, 교과 앱, 소개 페이지
- 최근 추가한 앱 3개 표시
- 검색어, 과목, 학년 필터로 앱 찾기
- 앱 카드와 `앱 열기` 버튼을 통한 새 탭 열기
- URL이 비어 있는 앱은 `준비 중`으로 표시
- 모바일, 크롬북, 노트북에서 보기 좋은 반응형 카드 UI

## 실행 방법

```bash
npm install
npm run dev
```

## 배포 전 확인

```bash
npm run build
```

Vite 기반 React 프로젝트라서 Vercel에 GitHub 저장소를 연결하면 바로 배포할 수 있습니다.

## 앱 추가 방법

앱 목록은 `src/data/apps.js` 파일에서 관리합니다. 새 앱을 추가하려면 `apps` 배열에 객체를 하나 더 넣으면 됩니다.

```js
{
  id: "new-app-id",
  title: "앱 제목",
  description: "앱을 한 줄로 설명합니다.",
  subject: "사회",
  grade: "5학년",
  tags: ["태그1", "태그2"],
  url: "https://example.com"
}
```

- `id`: 다른 앱과 겹치지 않는 고유한 값
- `subject`: 국어, 수학, 사회, 과학, 영어, 도덕, 실과, 음악, 미술, 체육, 창체 중 하나
- `grade`: 1학년, 2학년, 3학년, 4학년, 5학년, 6학년, 공통 중 하나
- `tags`: 검색에 함께 사용되는 키워드 배열
- `url`: 앱 주소. 비워두면 카드에 `준비 중`으로 표시됩니다.

최근 추가한 앱 3개는 `apps.js` 배열의 앞쪽 3개를 기준으로 홈 화면에 표시됩니다.
